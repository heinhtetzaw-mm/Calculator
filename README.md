# Calculator 🧮

A simple and responsive calculator built with JavaScript that performs basic arithmetic operations with a clean UI.

## Features
- Basic operations (Addition, Subtraction, Multiplication, Division)
- Real-time input handling
- Clear and delete functionality
- Responsive and modern design

## Tech Stack
- HTML
- CSS
- JavaScript
